Oracle Database 23ai is the next generation of the world's leading converged database.
It brings AI to your data with Oracle AI Vector Search, enabling you to store vectors
and run similarity searches natively.

Oracle AI Vector Search is built for Retrieval-Augmented Generation (RAG). It allows
you to quickly find documents that are semantically similar to a user's question,
without moving your data to a separate vector database.

By combining relational data, JSON, spatial, graph, and now vectors in a single engine,
Oracle Database 23ai simplifies data architecture. Developers can build sophisticated
AI applications using standard SQL and existing drivers.
